## Local Development
### Prerequisites
- [Python3](https://www.python.org/downloads/)
- [Node.js](https://nodejs.org/en)
- [NPM](https://www.npmjs.com/)

### Updating the Dataset
```python
python3 src/data/retrieve.py
```

### Running Locally
```sh
npm run start
```

### Deploy New Build (Publish)
```sh
npm run deploy
```
